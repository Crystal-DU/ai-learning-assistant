# AI Content Assistant

A modern Flask-based content extraction and prompt generation tool.

Extract content from files, URLs, or manual input, review and edit the extracted text, then generate structured AI prompts for ChatGPT or other LLMs.

---

## Features

### Content Import
- TXT files
- Markdown files
- PDF documents
- PowerPoint presentations (.pptx)
- Web page URLs
- Manual text input

### Content Processing
- Automatic content extraction
- Web page text scraping
- PDF text extraction
- PPT slide text extraction
- Editable preview area

### Prompt Generation
- Structured AI prompt creation
- Learning summary templates
- One-click copy

### User Interface
- Modern dashboard layout
- Responsive design
- Source information display
- File type detection
- Character counter
- Word counter
- Line counter
- Reset All button
